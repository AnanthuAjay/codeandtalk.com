Given how difficult it is to build performant hybrid mobile apps using web technologies for various platforms. React Native takes a different approach without promising you to write once and running everywhere. In this talk through how to build a React Native app using the shared knowledge but targeting different platforms.

In this talk I will walk through building a React Native app as we try to see how it solves this problem. Will cover the very basics, from how to get started all the way to avoid common performance pitfalls.
