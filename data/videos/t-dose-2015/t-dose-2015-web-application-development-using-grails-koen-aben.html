So you've heard Grails is a popular web-application framework for the JVM? Did you know you can create awesome web applications very rapidly because of its dynamic features, convention-over-configuration and sensible defaults? In this session the speakers will show you the power of Grails, by leading you through a sample application.

We'll be covering:

  • Focus will be on newest features in Grails 3, including Gradle and Spring Boot
  • Cool Web Development tricks using Groovy: a dynamic, multifunctional language on the Java Platform
In the end you have experienced the Fun and Joy in developing with Groovy and Grails!
